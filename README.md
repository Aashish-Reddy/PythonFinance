PythonFinance
=============

Requirements
------------

1. Python 3
2. numpy
3. scipy
4. pandas

Features
--------

1. Data Manager (Yahoo! Finance)
2. Basic Calculator Utils
3. Basic Date Utils no handle open NYSE Dates
4. Market Simulator
5. Event Study

  2 Event Finder
  3 Multiple Events (Past)
